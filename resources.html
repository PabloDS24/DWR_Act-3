Recursos para realizar la actividad

html

HyperText Markup Language

El código HTML es el lenguaje fundamental que utiliza un sitio web para estructurar y organizar su contenido. Actúa como el esqueleto del sitio, definiendo la disposición de los elementos como texto, imágenes, enlaces y otros elementos multimedia.

+Info

css

Cascading Style Sheets

El código CSS, o Hojas de Estilo en Cascada, determina la presentación visual y el diseño de un sitio web. Permite personalizar aspectos como el color, la tipografía, el espaciado y la disposición de los elementos HTML, mejorando así la apariencia y la experiencia del usuario en el sitio mientras navega a través de el.

+Info

javascript

JavaScript

JavaScript es un lenguaje de programación que los desarrolladores utilizan para hacer páginas web interactivas. Desde actualizar fuentes de redes sociales a mostrar animaciones y mapas interactivos, las funciones de JavaScript pueden mejorar la experiencia del usuario de un sitio web.

+Info

bootsrap

Bootstrap

Bootstrap es una biblioteca de herramientas de código abierto optimizadas para el diseño de sitios y aplicaciones web.

visual studio code

Visual Studio Code

es un editor de código fuente desarrollado por Microsoft para Windows, Linux, macOS y Web. Incluye soporte para la depuración, control integrado de Git, resaltado de sintaxis, finalización inteligente de código, fragmentos y refactorización de código.

+Info

github

Github

GitHub es un servicio basado en la nube que aloja un sistema de control de versiones (VCS) llamado Git. Éste permite a los desarrolladores colaborar y realizar cambios en proyectos compartidos, a la vez que mantienen un seguimiento detallado de su progreso.
